# TFI101_25_陳正勳_104人力銀行_關鍵字搜尋爬蟲

### 使用方式
在命令列使用
```
python 104.py arg1(關鍵字) arg2(前n頁)
```
eg. 
```
python 104.py 軟體 3
```
代表到104爬取關鍵字為軟體的前3頁

### 資料儲存(Exxcel)
欄位：更新日期、職缺連結、職缺名稱、公司名稱、公司連結、所屬產業、所在區域、工作經歷、學歷要求、職缺描述
檔名：104_{關鍵字}_first {最大頁數} pages_{建立時間}.xlsx
eg.
```
104_軟體_first 3 pages_2021-11-09 14-13-50.xlsx
```

### 程式執行資訊
執行完成後，會顯示Done
主程式執行過程錯誤可參見err.log
